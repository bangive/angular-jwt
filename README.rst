Angular-JWT
===========

Test app to handle auth between AngularJS and Flask-JWT, to be transformed in a real angular module later.

Requirements
============

- Backend : Flask and Flask-JWT
- Frontend : Angular, ui-bootstrap, angular-cookies

==========
